## 문제를 푸는 과정

1. 문제를 꼼꼼히 이해하기
2. 문제해결에 대한 계획 세우기
3. 계획에 오류가 있는지 확인하기
4. 계획 수행하기
5. 풀었다면 해결에 필요한 단서, 틀렸을 시 틀린 이유에 대해 피드백
